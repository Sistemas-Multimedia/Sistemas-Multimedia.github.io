1 Objective

This task shows how to implement a very simple lossy (gray-scale) image compressor [1] based on Scalar Quantization [4, 5, 6] and PNG [3]. This compressor basically removes the number of different gray tones in the image (normally, humans hardly distinguish more than 64 different colors). Therefore, the only source of “redundancy” exploited is the visual redundancy.

2 Scalar quantization applied to the “intensity” domain

The notetooks gray_SQ_deadzone.ipynb, gray_SQ_LloydMax.ipynb, gray_SQ_companded.ipynb, gray_SQ_midrise.ipynb, gray_SQ_midtread.ipynb and compare_quantizers.ipynb) shows the performance of different scalar quantizers using the quantization step sizes [4] \(\Delta \in \{128, 64, 32, 16, 8, 4, 2, 1\}\)1 Notice that, since the quantization step sizes are predetermined, no rate or distortion control [2] has been incorporated into the compressor.2

3 What do I have to do?

  1. Implement a RD control: Create a new notebook named “bit-rate_control.ipynb” that, given a maximum bit-rate \(R^\text {max}\), finds the quantization step \(\Delta ^*\) that minimizes the distortion. Use the quantizer [4] that your prefer.
