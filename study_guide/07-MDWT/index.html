1 Description

The 2D-DWT can a used on sequences of frames (images), by simply iterating as it is described in the Algorithm 1.1 [1]. N is the number of frames in the sequence V .

Algorithm 1.1: MDWT(V )
for i ← 0 to N − 1  do  V i = 2D-DWT(V i)
