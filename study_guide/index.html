1 Methodology

SM (Sistemas Multimedia) follows the PBL (Project-Based Learning) methodology. The students, helped by the teacher, develop a project during the course. This project is PRMC, a video compressor.

The project is implemented as a sequence of milestones.

2 Milestones

  1. OS (Operating System) Provisioning.
  2. Installation and basic programming with Python.
  3. Git, GitHub and the Fork-and-Branch Git Workﬂow.
  4. Motion Discrete Wavelet Transform.
