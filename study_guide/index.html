1 Methodology

SM (Sistemas Multimedia) follows the PBL (Project-Based Learning) methodology. The students, helped by the teacher, develop a project during the course. This project is PRMC, a video compressor. In the Teaching Guide you can ﬁnd the details about the evaluation of the subject.

The project is described as a sequence of milestones (or topics), each one with the following structure:

  1. A description of the milestone. This part is worked in class.
  2. A What you have to do? section, which describes the virtualized job of the students.
  3. How much time you should spend working in the previous virtualized section.
  4. The deliverables that you should generate and send to the teacher.
  5. A list of bibliographic references.

2 Milestones

  1. OS (Operating System) Provisioning.
  2. Installation and basic programming with Python.
  3. Git, GitHub and the Fork-and-Branch Git Workﬂow.
  4. Motion Discrete Wavelet Transform.
