1 Description

(Digital) Images are 2D (discrete in space and amplitude) signals. The 2D-DWT of an image can be computed using (1) separable 1D ﬁlters, and (2) nonseparable 2D ﬁlters. Except in very special cases, all 2D-DWT implementations use separable ﬁlters.

We deﬁne

{lls+1,lhs+1,hls+1,hhs+1} = 2D-DWT(lls), (1)

where lls is a 2D-signal, and

{{{ls+1} i,{hs+1}} i} = {DWT(llis)}, (2)

is the result of using the 1D-DWT to each row of lls, generating a sequence of low-pass subbands {ls+1} and sequence of high-pass subbands {ls+1}, and

2D-DWT(lls) = {DWT({{{ls+1} i,{hs+1}} i})}, (3)

is the result of using the 1D-DWT to each column of the {{{ls+1}i,{hs+1}}i} structure.
