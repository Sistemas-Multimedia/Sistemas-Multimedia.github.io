1 Description

inputs data, which basically is a sequence of (digital) images. A image is a matrix of pixels. This milestone shows the structure of the videos used in .

We are going to store a video as a sequence of ﬁles with names <preﬁx>000.png, <preﬁx>001.png, ..., where <preﬁx> speciﬁes the video name and “.png” indicates that the images are coded using the PNG image format. All the images of a video have the same spatial resolution, are represented in the RGB domain, and each color component has bit depth of 16 bits, representing an unsigned integer. The pixel intensities are displaced to the center of the range of values ([0, 65535]) that the components can take (for example, in disk, the component intensity 0 is represented by 32768).
