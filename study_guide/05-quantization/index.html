1 Description

Humans are quite eﬃcient recognizing the information stored in images (and frames of a video), even when this information has been degraded or (partially) lost. Quantization [1, 3] is a technique that can remove the part of the visual information that is less relevant for us, and implies lossy coding, which provides compression ratios usually at least one order of magnitude higher than using lossless coding.

In this milestone we are quantize a frame in the RGB color domain. A quantizer (typically denoted by Q) is a encoding system that inputs a sequence of (digital) samples x and output a sequence of quantization indexes k (see the Fig. 1). The inverse system, called a dequantizer (and denoted by Q−1), recovers an approximate version of x that we will denote as x̃, whose similarity with x inversely depends on a frame quantization step (QS) Δ. Notice that although Δ is an input parameter to the quantizer, it has not been considered in the ﬁgure to keep it simpler.

We deﬁne the quantization error1

ei = xi −x̃i, (1)

that obviously, should be minimized.


Figure 1: Scalar quantization and dequantization of a signal.

Quantizers can be classiﬁes into:

  1. Scalar quantizers: those that produce a quantization index ki by each input sample xi.
  2. Vector quantizers: that process the input by blocks of samples, called vectors, producing a quantization index by vector, and usually, the length of the quantization index y much shorter than the length of the vector. Vector Quantization (VQ) can remove correlation and therefore, are more eﬃcient that Scalar Quantization (SQ). Unfortunately, the computational requirements of VQ are, by far, higher than the needed by SQ. If we also consider that there are other techniques (such as transform coding) that are able to decorrelate the samples with less computation than VQ, we can understand why SQ has been selected, for example, in most video codecs.

Quantizers, also, can be classiﬁed as:

  1. Uniform quantizers: those in which Δ is idependent on the amplitude of the samples.
  2. Non-unifom quantizers, on the contrary, when Δ depends (directly or indirectly) on the amplitude of the samples.

Non-uniform quantizers can be:

  1. Static quantizers: if Δ depends on the amplitude of the samples (as happens, for example, in companded quantizers), but this dependency is known a-priori.
  2. Adaptive quantizers: when Δ is adapted to minimize the quantization error, depending “on-the-ﬂy” on the characteristics2 of the signal.

All quantizers can be classiﬁed into:

  1. Mid-tread quantizers, if x̃i can be 0.
  2. Mid-rised quantizers, if x̃i never is 0, even if xi = 0.

Finally, quantizers can be classiﬁed as:

  1. Dead-zone quantizers, that are characterized by a QS of length 2Δ for xi = 0. Deadzone quantizers tends to remove the electronic noise (that usually has an small amplitude compared to the input signal x). Notice that dead-zone quantizers should not be consdered uniform, and that all dead-zone quantizers, by deﬁnition, are mid-tread.
  2. No dead-zone quantizers, when the dead-zone does not exist.

1.1 Quantization in the RGB domain

Supposing that we will use a static uniform dead-zone quantizer with quantization steps, a color RGB image is quantized chanel by channel, using QSs ΔR, ΔG, and ΔB. A reasonable question here is, how we should be chosen these parameters? At this point we can consider two diﬀerent optimization perspectives. In the ﬁrst one, we consider strictly visual considerations, and obviously, any alternative diﬀerent from

ΔR = ΔG = ΔB (2)

will produce some alteration in the color (sometimes called “chroma”) of the reconstructed frame.

In the second one, only the (RD performance is considered. From a RD perspective, the best combination of QSs is those that optimizes (makes it closer to the origin of coordinates) the RD curve. Thus, the optimal QSs should operate in the curves with the same RD slope,

λR = λG = λB, (3)

for a given total bit-rate R, which implies that the contribution of each channel (the ratio between quality and bit-rate) to the quality of x̃ has been highest possible [2, 1].
