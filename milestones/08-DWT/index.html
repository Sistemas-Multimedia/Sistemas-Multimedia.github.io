1 The transform

Technically, the DWT (Discrete Wavelet Transform) is a linear basis expansion which computes a critically-sampled octave-band decomposition [4, 1]. As the rest of transforms:

  1. The output of the DWT describes the (input) signal in a set of subbands, producing the so called a decomposition.
  2. The index of the subband is related to the frequency of the signal. For example, in the case of the images, the position of the coeﬃcients in the subbands is related to the spatial area where the corresponding pixels are found.

1.1 1D case

The 1D-DWT convolves the input signal with a set of basis functions (you can see the basis functions of several DWT in the Wavelet Browser) deﬁned by a cascade of ﬁlters banks, building a dyadic signal decomposition in the frequency domain.

1.1.1 An example of DWT: the Haar DWT

The DWT is also described by the ﬁlters of the analysis matrix transform (the matrix K in previous milestones). Haar deﬁned the analysis downsampling ﬁlters

[ lns+1 hns+1 ] = [ 1 2 1 2 1 2 −1 2 ] [ l2ns l2n+1s ] , (1)

where the superindex s denotes the subband, and n refers to the n-th element of the signal. By deﬁnition (and notice that this holds for all DWTs),

l0 = x. (2)

As it can be seen in the Fig. 1, K0 is a low-pass ﬁlter and K1 is a high-pass ﬁlter (this holds for all DWTs). Considering the Haar ﬁlter, we can conclude that:

  1. There exists aliasing between the ﬁlters (this is also true for all DWTs) and this is a drawback because:
    1. The same information can be found in both subbands (l and h). Therefore, the concentration of the energy in one of the subbands is smaller.
    2. At it can be seen also in Eq. 1, the subbands are downsampled by 2 (action usually represented by ↓ 2) and both subbands should have, at most, a bandwidth of 1 2 radians/sample in order to avoid the aliasing during the subsampling, thus increasing the perceptible quality of the low-pass subband l.

    These problems can be only solved using ﬁlters that have transfer functions that overlaps a smaller area1 (see Figs. 2 and 3).

  2. There is not phase distortion (the phase of the ﬁlters is a linear function of the frequency, see the Fig. 4). This means that both coeﬃcients lns+1 and hns+1 refers to the same section of signal {l2ns,l2n+1s}, allowing the design of entropy codecs for the decomposition which exploit the correlation between subbands. Notice that this does not hold for all ﬁlters (see Fig. 5).


Figure 6: Computation of the S-levels DWT and the generated subbands.

Eq. 1 computes the 1-levels Haar DWT. A more general expresion for this equation (and this holds for all DWTs) is

{ls+1,hs+1} = DWT(ls), (3)

where {⋅,⋅} denotes the concatenation of subbands. As can be seen in the Fig. 6, it’s possible to compute the S-levels DWT, using Eq. 3 iteratively over the low-frequency subband, generating the decomposition

{l0S,h 0S,h 0S−1h 1S−1,h 0S−2h 1S−2h 2S−2h 3S−2,⋯,h 01h 11⋯h 2n−1−11} = DWTS(l0), (4)

where

n = log ⁡ 2(N) (5)

where N is the number of samples.

In the case of the Haar ﬁlters, the synthesis (inverse) 1-levels DWT (that we will denote by DWT−1) is the result of solving the coeﬃcients lns+1 and hns+1 in the Eq. 1. In general, for longer DWT ﬁlters we are going to have more coeﬃcients, but the same procedure can be used for ﬁnding the synthesis transform. Therefore, it can be written that

ls = DWT−1({ls+1,hs+1}), (6)

and DWT−S, the S-levels synthesis DWT, can be computed as it is also described in the Fig. 6, by simply reversing the analysis steps.

1.1.2 Multiresolution analysis

In the decomposition described in Eq. 4 there are S + 1 subbands, and therefore, it is possible to compute S + 1 resolution levels of the analyzed signal l0:

  • lS: the resolution level S of l0 (no DWT−1 has been applied in the synthesis process), with have only one sample l0S that uses to be the arithmetic mean of l0. The coeﬃcient l0S is also called the DC (Direct Current) coeﬃcient. On the contrary, the rest of coeﬃcents (high-frequency subbands) are called the AC (Alternating Current) coeﬃcients.
  • lS−1 = DWT−1({lS,hS}), the resolution level S − 1 of l0, with two samples.
  • lS−2 = DWT−1({lS−1,hS−1}), the resolution level S − 1 of l0, with four samples.
  • ⋮ ⁡
  • The resolution level 0, l0 = DWT−1({l1,h1}).

1.1.3 Filters


Figure 7: A 2-channels PRFB (Perfect Reconstruction Filter Bank). See this notebook.

All DWTs are deﬁned by 4 ﬁlters (using a similar notation that the followed by PyWavelets) (see Fig. 7):

  1. Analysis (downsampling) low-pass ﬁlter ϕ~ (the decomposition scaling function), which computes ls+1 from ls.
  2. Analysis (downsampling) high-pass ﬁlter ψ~ (the decomposition wavelet function), which computes hs+1 from ls.
  3. Synthesis (upsampling) high-pass ﬁlter ϕ (the reconstruction scaling function), which computes {l2ns} from ls+1 and hs+1.
  4. Synthesis (upsampling) high-pass ﬁlter ψ (the reconstruction wavelet function), which computes {l2n+1s} from ls+1 and hs+1.

In the context of the DWT, orthogonality provides intersubband decorrelation, which basically means that the contribution of each subbands to the reconstruction (and this holds for all DWTs) of the signal are independent. Orthogonal ﬁlters can be recognized because:

  1. ϕ~⊥ψ~ and ϕ⊥ψ, where ⊥ denotes orthogonality.
  2. Their taps are all diﬀerent (the ﬁlters are asymmetric).
  3. The taps of the synthesis ﬁlters can be determined simply by “reﬂecting” the taps of the analysis ﬁlters.
  4. In general, except for the Haar DWT, orthogonal ﬁlters generate phase distortion.

On the other hand, in a biorthogonal DWT:

  1. ϕ~⊥ψ~, ϕ⊥ψ, ψ⊥ϕ~ and ψ~⊥ϕ.
  2. All the ﬁlters are symmetric.
  3. The ﬁlters satisfy that
    ψ = (−1)nϕ~; n ∈ ℕ, (7)

    and

    ϕ = (−1)nψ~; n ∈ ℕ (8)

    (the odd taps are multiplied by − 1). Therefore, if we have the analysis ﬁlters we can deduce the synthesis ﬁlters and viceversa.

  4. No phase distortion is generated.
  5. In orthogonal transform, all the subbands have the same gain, but this is not true for biorthogonal DWTs.

1.2 2D case

(Digital) Images are 2D (2-Dimensional, discrete in space and amplitude) signals. The 2D-DWT of an image can be computed using (1) separable 1D ﬁlters, and (2) nonseparable 2D ﬁlters  [3]. Except in very special cases, all 2D-DWT implementations use separable ﬁlters by simplicity. The Figure 8 show an example of basis functions used by a 3-levels 2D-DWT.


Figure 8: 2D-DWT basis functions (see this notebook).

Separability in the DWT context means that we can compute the 1-levels 2D-DWT using the 1D ﬁlters, by applying them to each dimension, and using in-place operations. This procedure has been described in the Algorithms 1.1, 1.2 and 1.3, where Xr,∗ refers to the r-th row of the matrix X and X∗,c to the c-th column, being R and C the number of rows and columns of the input image X. See also the Fig. 9.

Algorithm 1.1: Rows-DWT(X)
for r ← 0 to R − 1  do Xr,∗ = DWT(Xr,∗) return (X)

Algorithm 1.2: Columns-DWT(X)
for c ← 0 to C − 1  do X∗,c = DWT(X∗,c) return (X)

Algorithm 1.3: 2D-DWT(X)
X = Rows-DWT(X) X = Columns-DWT(X) return (X)

As it can be seen in the Fig 9, the obtained 2D decomposition is expressed by

[ LL HL LH HH ] = 2D-DWT(X), (9)

where L stands for low-pass ﬁltering and H for high-pass ﬁltering. Notice that LL = LL1, LH = LH1, HL = HL1, and HH = HH1.

Eq. 9 describes the 1-levels (analysis) 2D-DWT. Replacing LL = Ls+1, X = Ls, and {HL,LH,HH} = Hs+1 in the previous expression, the S-levels 2D-DWT of L0 = X (2D-DWTS(L0)) can be computed applying S-times

{Ls+1,Hs+1} = 2D-DWT(Ls) (10)

to the low-frequency subband Ls. As an example, the Fig. 10 shows the 3-levels 2D-DWT of the image lena.

1.4.1 Quantization

In the case of biorthogonal transforms, and in absence of RD optimization, the quantization steps uses for each subband should be inversely proportional to the L2 synthesis gain of the subbands2  [2]. However, notice that in orthogonal DWTs all the subbands have the same gain.

1.4.2 Quantization

As usually, we can use

Δ0 = Δ1 = ⋯ = ΔN−1, (12)

where Δi is the quantization step used for the i-th frame of the sequence. However, notice that this quantization pattern does not necessaryly optimizes the RD tradeoﬀ. The RD curves of each frame should be taken into consideration in order to perform a good rate-control.
