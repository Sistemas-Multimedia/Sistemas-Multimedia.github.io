2 What you have to do?

Please, create an experiment in which the frame quantization step is diﬀerent for each frame of a sequence, and another experiment that uses the same quantization step for all the frames. Compute the average RD curves. Which one is better? Do you think that

D = ∑ iDi,

where D is the distortion generated for the complete sequence and Di is the distortion only for the i-th frame, holds?
