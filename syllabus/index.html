Sistemas Multimedia - Syllabus

Vicente González Ruiz

July 7, 2022

1 About Sistemas Multimedia

This course introduces and develops some of the most used techniques used in multimedia systems. The teaching methodology is based in the project based learning and such techniques are applied to develop an open-source video compressor: MVRC (Multi-Resolution Video Codec).

2 Milestones

The development of the MRVC project is organized in a collection of milestones. Some of them has been previously developed and our goal is to understand how the implemented techniques work and how to use them. The rest of milestones propose the implementation of new features in MRVC.

Milestone 1: Preparing the framework.

The working environment used in Sistemas Multimedia is based on Linux, Python and Git.

Task 1: Operating system provisioning.

Task 2: Installation and basic programming with Python.

Task 3: Git, GitHub and the fork-and-branch Git workflow.

 

Milestone 2: Entropy coding of images:

Entropy is a measure of the amount of information represented by a piece of data. Most of image and video compressors (and MRVC is not an exception) achieve a reduction in the amount of output data via entropy coding. This milestone analyzes some of the most used lossless image compressors.

Task 1: PNG (Portable Network Grachics).

Task 2: JPEG (Joint Photographic Expert Group) 2000. TO-DO.

Task 3: JPEG XL. TO-DO.

Task 4: BPG (Better Portable Graphics). TO-DO.

 

Milestone 3: Scalar quantization:

Raw images and videos contain large amounts of information that can be removed without generating a meaningful visual distortion. Quantization is the basic technique that allows to delete information in images and videos (and in signals in general).

Task 1: Scalar quantization of gray-scale images.

Task 2: Scalar quantization in the \(\text {RGB}\) color domain. (Partially TO-DO)

Task 3: Trellis Coded Quantization (TCQ) of gray-scale images. TO-DO.

 

Milestone 4: Exploiting the RGB color (mainly visual) redundancy:

In color images, the RGB (Red, Green, and Blue) channels tend to be highly correlated. We can take advantage of this channel correlation to increase the compression ratio in image and video coding, without generating noticeable visual artifacts.

Task 1: \(\text {RGB}\) Decorrelation with the DCT (Discrete Wavelet Transform).

Task 2: \(\text {RGB}\) Decorrelation with the YCoCg (luminance, orange chroma, green chroma) Transform.

Task 3: \(\text {RGB}\) Decorrelation with the YCrCb (luminance, red chroma, blue chroma) Transform.

Task 4: \(\text {RGB}\) Decorrelation with the RCT (Reversible Color Transform) domain. TO-DO.

Task 4: \(\text {RGB}\) Decorrelation with the LMS (Long, Medium, Short) Transform. TO-DO.

Task 5: Vector Quantization of the \(\text {RGB}\) color domain.

 

Milestone 5: Exploiting spatial redundancy in images:

Visual data is highly correlated in the spatial domain and such correlation can be considered to achieve higher compression ratios.

Task 1: Spatial Decorrelation with the DCT.

Task 2: Spatial Decorrelation with the DWT (Discrete Wavelet Transform).

Task 3: Vector Quantization in the Spatial Domain.

 

Milestone 6: Exploiting temporal redundancy in the visual domain:

Most image sequences (videos) are highly correlated in the temporal domain and motion estimation can exploit this redundancy.

Task 1: Motion estimation in the image domain.

Task 2: Motion compensation in the image domain.

 

Milestone 7: Removing temporal redundancy in the transform domain:

Some transforms facilitate the multiresolution representation of images and this can be interesting to estimate and compensate the motion in this domain.

Task 1: Motion compensation in the DWT domain.

Task 2: Motion compensation in the DCT domain.

Task 3: Motion estimation and compensation in the LP (Laplacian Pyramid) image domain.

 

Milestone 8: MRVC (Muti-Resolution Video Coding):

Cooking all the ingredients together in a video compressor.

Task 1: MVRC.
