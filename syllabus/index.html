1 About this course

This course shows the basic principles used to encode digital images and videos, and how to apply them in commercial and scientiﬁc environments.

Images and sequences of images (videos) are everywhere, and one of the main problems that arise when we use them is the huge amount of data that they require. This course introduces the theoretical contents based on the development of a video encoding project. Therefore, it is an imminently practical course, organized into a collection of milestones.

2 Timetable

Week 1: Preparing the working environment.
We will use a Linux operating system with Git and Python to develop our project.
Milestone 1: Operating system provisioning.
Milestone 2: Installation and basic programming with Python.
Milestone 3: Git, GitHub and the fork-and-branch Git workﬂow.
Milestone 4: Accesing to the MRVC project.

 

Week 2: Removing spatial redundancy.
Pixels are spatially correlated, and human beings prefer low spatial frequencies to high ones.
Milestone 4: Entropy coding of images with PNG (Portable Network Graphics).
Milestone 7: DCT (Discrete Cosine Transform) (TO-DO).
Milestone 8: DWT (Discrete Wavelet Transform).
Milestone 8: Removing Redundancy with the 2D-DWT.
Milestone 8: The Motion Discrete Wavelet Transform.
Milestone 9: LP (Laplacian Pyramid).

 

Week 3: Removing color redundancy.
Color components are correlated, and human beings are more sensitive to slow variations of the color than to high frequency changes.
Milestone 5: Quantization in the RGB domain.
Milestone 6: Quantization in the YUV domain.

 

Week 4: Removing temporal redundancy.
Milestone 10: Motion estimation and compensation.

 

Week 5: IBP... coding in the image domain (TO-DO).

Week 6: MCTF (Motion Compensated Temporal Filtering) in the imagen domain (TO-DO).
 
Week 7: IPP... coding in the DWT/LP domain.

Week 8: IBP... coding in the DWT/LP domain (TO-DO).
