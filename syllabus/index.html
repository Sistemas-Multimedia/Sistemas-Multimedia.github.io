Vicente González Ruiz

November 21, 2021

1 About this course

This course analyses the basic principles used to encode digital images and videos, and how to apply them in commercial and scientiﬁc environments.

Images and sequences of images (videos) are everywhere, and one of the main problems that arise when we use them is the huge amount of data that they require. This course introduces the theoretical contents based on the development of a video encoding project. Therefore, it is an imminently practical course, organized into a collection of milestones.

2 Timetable

Day 1: Preparing the working environment.
We will use a Linux operating system with Git and Python to develop our project.
Milestone 1: Operating system provisioning.
Milestone 2: Installation and basic programming with Python.
Milestone 3: Git, GitHub and the fork-and-branch Git workﬂow.

 

Day 2: Compressing color images.
Quantization and entropy coding, two key compression techniques.
Milestone 4: Entropy coding of images with PNG.
Milestone 5: Quantizing in the RGB domain.

 

Day 3: Removing color redundancy.
Color components are correlated, and human beings are more sensitive to slow variations of the color than to high frequency changes.
Milestone 6: Quantizing in the luma/chroma domain.

 

Day 4: Removing spatial redundancy with transforms I.
Quantization in the transform domain is more eﬃcient than in the image domain.
Milestone 7: The DCT (Discrete Cosine Transform).

 

Day 5: Removing spatial redundancy with transforms II.
Providing spatial multiresolution to image coding.
Milestone 8: The DWT (Discrete Wavelet Transform).

 

Day 6: Removing temporal redundancy with motion estimation.
Video sequences are correlated in time, and determining how the objects move can help to compress more.
Milestone 9: Motion estimation and compensation in the image domain.
Milestone 10: IPP... coding in the image domain.

 

Days 7: MRVC (Muti-Resolution Video Coding).
Providing spatial multiresolution to video coding.
Milestone 11: Motion compensation in the transform domain.
Milestone 12: IPP... coding in the transform domain.
