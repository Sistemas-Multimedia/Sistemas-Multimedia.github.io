1 About Sistemas Multimedia

This course introduces and develops some of the most common multimedia processing techniques. The teaching methodology is based in the project based learning and such techniques are used to develop an open-source video compressor: MVRC (Multi-Resolution Video Codec).

2 Milestones

The development of the MRVC project is organized in a collection of milestones. Some of them has been previously developed and our goal is to understand how the implemented techniques work and how to use them. The rest of milestones propose the implementation of new features in MRVC.

Milestone 1: Preparing the working environment.

In most cases, open source frameworks increase the portability of the software and eases the deployment!

Step 1: Operating system provisioning.

Step 2: Installation and basic programming with Python.

Step 3: Git, GitHub and the fork-and-branch Git workﬂow.

 

Milestone 2: Removing the color redundancy:

In color images, the RGB channels tend to be highly correlated. We can take advantage of this channel correlation, for example, to decrease the entropy and therefore, increase the compression ratio in image and video coding.

Task 4: The 1D-DCT (1-Dimensional Discrete Cosine Transform) transform applied over the color domain.

Task 5: The YCrCb transform.

Task 6: The YCoCg transform.

 

Milestone 3: Removing the spatial redundancy:

Visual data is highly correlated in the spatial domain and such correlation can be exploited to decrease the entropy in images.

Task 7: 2D-DPCM (2-Dimensional Diﬀerential Pulse Code Modulation).

Task 8: The 2D-DCT (2-Dimensional Discrete Cosine Transform).

Task 9: The 2D-DWT (2-Dimensional Discrete Wavelet Transform).

 

Milestone 4: Entropy coding:

So far, the minimization of the entropy has been our main objective. Let’s see how can we use an entropy codec to build a lossless (or near lossless, depending the accuracy of the transforms) image compressor.

Task 10: PNG (Portable Network Grachics).

Task 11: JPEG (Joint Photographic Expert Group) 2000 as an entropy codec.

Task 12: CBAC (Context-Based Arithmetic Coding).

 

Milestone 5: Rate-control through quantization:

In most images, a large percentage of the code-stream is dedicated to encode inappreciable (by humans) visual information. Quantization can be used to remove such information, providing higher compression ratios.

Task 13: Quantization in the transform domain.

Task 14: RDO (Rate/Distortion Optimization).

 

Milestone 6: Perceptual coding through chroma “subsampling”:

Humans hardly perceive high frequency contents in the chroma components.

Task 15: Chroma ﬁltering and subsampling.

 

Milestone 7: Motion estimation and compensation in the image domain:

Most image sequences are highly correlated in the temporal domain and motion estimation can exploit this redundancy to, again, increase the compression ratio in video coding.

Task 16: Motion estimation in the image domain.

Task 17: Motion compensation in the image domain.

 

Milestone 8: MRVC (Muti-Resolution Video Coding):

It’s possible to provide multiresolution (spatially-scalable) code-streams by estimating and compensating motion in the transform domain.

Task 16: Motion estimation and compensation in the transform domain.

Task 17: Symmetric video coding.
